# インタプリタ
## 目的/概要
* ある用途に対し、新しい言語を作ることで問題を解決するもの。

## 構成
以下の4つのもので構成されている。

* AbstractExpression(抽象表現): 共通のインターフェースを定義
* TerminalExpression(終端): 終端を表現するクラス
* NonterminalExpression(終端以外): 非終端を表現するクラス
* Context(状況,文脈): 構文の解析を手助けする
